# 顶级股票交易系统 v2.0 精简实战版

> 理论指导实践，纪律决定成败。本版聚焦可执行规则，删除低价值理论，强化实战适配。

## 一、核心哲学与系统架构

### 1.1 核心原则

| 原则 | 内涵 |
|------|------|
| **概率思维** | 不追求每笔必赢，追求正期望值的长期复利 |
| **风险管理优先** | 先求不败，后求胜，保护本金是第一要务 |
| **系统化执行** | 规则驱动，消除情绪干扰，可重复优于可解释 |
| **适应性进化** | 市场变则系统变，持续验证迭代 |

### 1.2 系统四层架构

```
交易系统
├── 分析层 —— 判断市场状态与机会
│   ├── 市场趋势判定（七维共振）
│   ├── 资金面与情绪面
│   └── 个股筛选（排雷 + 基本面 + 技术信号）
│
├── 决策层 —— 决定买卖什么、何时、多少
│   ├── 策略选择决策树（市场状态 → 策略映射）
│   ├── 入场/出场规则（ATR动态参数）
│   └── 仓位管理（风险预算 + 回撤恢复）
│
├── 执行层 —— 纪律性下单与跟踪
│   ├── 每日执行流程
│   └── A股特殊风险应对
│
└── 反馈层 —— 复盘与迭代
    ├── 交易日志与绩效归因
    └── 回测验证与系统调优
```

## 二、市场环境判断与策略选择

### 2.1 市场趋势状态（七维判定 v6）

> **核心原则：** 加权评分制，各维度按权重累加至牛/熊总分，结合状态惯性防止方向跳变。七维度覆盖动量、广度、超买超卖、杠杆资金、量价确认、外资方向、市场情绪。

| 维度 | 权重 | 指标 | 牛市信号 | 震荡信号 | 熊市信号 |
|------|------|------|---------|---------|---------|
| **MACD** | 1.0 | DIF/DEA(12,26,9) | 金叉 + 零轴上方 + 柱状图放大（背离×1.5） | DIF与DEA缠绕，柱状图缩短 | 死叉 + 零轴下方 + 柱状图放大（背离×1.5） |
| **涨跌家数** | 1.5 | 沪深上涨/下跌比 | ≥ 2.0（强牛2.25），≥ 1.5（牛1.5） | 涨跌各半 | ≤ 0.5（强熊2.25），< 0.67（熊1.5） |
| **RSI** | 1.0 | RSI(14) | > 60 且上行，或从超卖区(<30)回升（背离×1.5） | 40~60 区间震荡 | < 40 且下行，或从超买区(>70)回落（背离×1.5） |
| **融资余额** | 1.2 | 10日线性回归斜率 | 回归斜率 > +0.3%/日（> 0.5为强1.8） | 斜率 ±0.3% 以内 | 回归斜率 < -0.3%/日（< -0.5为强1.8） |
| **量价配合** | 1.3 | OBV 趋势 + 背离 | 价涨量增 / OBV底背离（强1.95） | 量价关系无明显异常 | 放量下跌 / OBV顶背离（强1.95） |
| **北向资金** | 1.5 | 5日/20日成交额均量比 | ≥ 1.2（≥ 1.4为强2.25） | 0.8~1.2 | ≤ 0.8（≤ 0.6为强2.25） |
| **涨跌停** | 1.3 | 并行评分（6因子） | net ≥ 2（强≥4→1.95） | -2 < net < 2 | net ≤ -2（强≤-4→1.95） |

**涨跌停维度（并行评分机制）**: 同时计算牛信号分（bullPts）和熊信号分（bearPts），互不干扰：

| 评分因子 | 牛信号加分 | 熊信号加分 |
|---------|-----------|-----------|
| 涨跌比 | ≥5:1 +3, ≥2.5:1 +2, ≥1.5 +1 | ≤0.2 +3, ≤0.4 +2, <0.8 +1 |
| 跌停绝对数 | — | ≥100 +3, ≥50 +2, ≥20 +1 |
| 涨停+封板质量 | ≥80且封板≥70% +3, ≥40且≥50% +2, ≥20且≥40% +1 | — |
| 打板次日收益 | ≥+2% +1 | ≤-2% +2, <-0.5% +1 |
| 低封板率 | — | 封板率<40%且涨停≥15 +1 |
| 低赚钱效应 | — | 赚钱效应<30% +1 |

net = bullPts − bearPts，net ≥ 4 为强牛，net ≥ 2 为牛，net ≤ -4 为强熊，net ≤ -2 为熊。

**加权评分与状态判定**:

```
牛信号总权重 bullW = Σ(各维度牛信号 × 该维度权重 × 强弱系数)
熊信号总权重 bearW = Σ(各维度熊信号 × 该维度权重 × 强弱系数)
net = bullW - bearW

状态判定（7维总权重 8.8 校准）：
  牛市：bullW ≥ 4.5 且 net > 0
  熊市：bearW ≥ 4.5 且 net < 0
  偏多：bullW ≥ 3.0 且 net > 0
  偏空：bearW ≥ 3.0 且 net < 0
  震荡：其他

确认：bullW 或 bearW ≥ 3.5 为 confirmed
```

**状态惯性（hysteresis）**: 防止相邻判定周期状态频繁跳变：
- 同方向调整（牛市↔偏多、熊市↔偏空）：自由切换
- 跨方向切换或进出中性：需 |net| ≥ 2.5，否则保持上一状态
- 前一状态持久化，页面刷新后恢复

> **回测验证（沪深300，2023-2024）：** 单维度准确率 50~54%，七维加权综合后准确率约 52~55%。纯技术指标在 A 股存在准确率天花板，需结合基本面与政策面综合判断。状态惯性机制可有效避免盘中短期波动导致的状态跳变。

**多头窗口快速判定：** 指数收盘站上MA60 + MA60连续3日拐头向上 + 当日涨跌比 ≥ 1.5。

### 2.3 策略选择决策树

> **核心规则：市场状态决定策略，策略决定仓位，不可错配。**

```
市场状态 → 策略选择
─────────────────────────────────────────
牛市确认  → 趋势突破（主策略），仓位上限 80-100%
偏多      → 回调买入（主策略），仓位上限 50-70%
震荡      → 回调买入（减仓版，仓位上限 50%）
偏空      → 仅观望，仓位上限 20-40%
熊市      → 空仓（底部右侧策略仅限模拟盘，仓位上限 0-20%）
```

| 策略 | 适用市场 | 持有期 | 时间止损 | 风险等级 | 收益预期 |
|------|---------|--------|---------|---------|---------|
| 趋势突破 | 牛市确认 | 1-4周 | 5日未突破前高 → 出场 | 中 | 高 |
| 回调买入 | 偏多/震荡 | 2-8周 | 8日未企稳（未回MA20上方）→ 出场 | 低-中 | 中-高 |
| 底部右侧确认 | 熊转牛初期 | 1-6月 | 15日未突破颈线 → 出场 | 高 | 极高 |

> **注意：** 底部右侧策略为高难度策略，实盘资金不超过总资金 3%（单只），建议模拟盘练习至少10次信号后再实盘。

## 三、选股体系（三层漏斗）

### 3.1 三层漏斗总览

```
全市场 (5000+)
    │
    ▼ 第一层：硬性排雷 + 基本面初筛（一票否决 + 定量）
    │
    ▼ 第二层：景气度验证（资金 + 业绩）
    │
    ▼ 第三层：技术面确认（买入信号）
       │
       最终标的池 → 盈亏比计算 → 买入
```

### 3.2 第一层：排雷 + 基本面初筛

**硬性排雷（任一触发直接排除）：**

**选股器自动筛：**
```
非ST且非停牌且非退市且商誉占净资产比例小于30%且大股东质押比例小于60%
且上市天数大于250天且最近一期审计意见为标准无保留意见
```

**F10 手动复核：**

| 排雷项 | 排除标准 |
|--------|---------|
| 连续亏损 | 连续2年净利润为负 → 排除（ST/退市风险） |
| 关联交易异常 | 关联交易/营收 > 40% → 排除 |
| 存贷双高 | 货币资金/总资产>30% 且 有息负债/总资产>30% → 排除 |
| 董监高减持 | 近一年累计减持 > 总股本5% → 排除 |
| 经营现金流异常 | 最近一年经营现金流为负 → 排除 |
| 诉讼/处罚 | 近一年重大诉讼或监管处罚 → 排除 |

**基本面定量筛选：**

| 维度 | 指标 | 标准 |
|------|------|------|
| 盈利能力 | ROE | > 12%（连续3年） |
| 成长性 | 营收/利润增速 | > 15% |
| 财务健康 | 资产负债率 | < 60%（非金融） |
| 现金流 | 经营现金流/净利润 | > 0.7 |
| 估值 | PEG | < 1.2 |
| 护城河 | 毛利率趋势 | 稳定或提升 |

**选股器一句话：**
```
ROE大于12%且营收同比增长大于10%且净利润同比增长大于10%且资产负债率小于60%
且经营现金流为正且市盈率5到40且流通市值大于30亿且商誉占净资产比例小于30%
且质押比例小于60%且非ST且非停牌
```

**估值锚定（仅保留实用方法）：**

| 方法 | 适用场景 | 标准 |
|------|---------|------|
| PE | 稳定盈利公司 | PE < 行业中位数 × 0.8 为低估 |
| PEG | 成长股 | < 1 低估，1-1.5 合理，> 1.5 高估 |
| PB | 银行、地产、周期股 | PB < 1 为安全边际 |

> 结果：约 200-400 只。

### 3.4 第三层：技术面确认

**趋势突破信号：**
```
今天放量突破20日高点 且 MACD金叉 且 MACD柱为正 且 20日均线向上 且 60日均线向上
```
> **回测验证：A股5年数据，信号胜率应 ≥ 45%，盈亏比 ≥ 2:1。**

**回调买入信号：**
```
股价在60日均线上方 且 60日均线向上 且 近5日缩量回调至20日均线附近 且 今日成交量为近20日最低
```

> **回测验证：震荡市胜率应 ≥ 50%。**

**底部右侧确认信号：**

```
股价从半年高点下跌超过40% 且 近5日成交量萎缩后今日放量 且 RSI从30以下拐头向上 且 MACD金叉
且 当日成交量 > 20日均量 × 2（放量确认）
```

## 四、买入与卖出规则

### 4.1 趋势突破买入法

```
入场条件（全部满足）：
1. 收盘价 > HHV(HIGH, 20) 且 当日成交量 > MA(VOL, 20) × 1.5
2. MACD(12,26,9) DIF > DEA 且 MACD柱状图 > 0
3. 大盘处于"牛市确认"或"偏多"状态
4. 基本面符合标准 + 未触发排雷清单
5. 盈亏比 ≥ 2:1

仓位分配（金字塔加仓）：
  首批：计划仓位 40%
  确认加仓：回踩确认不破突破位 → 加仓 30%
  趋势加速：放量创新高 → 再加仓 20%（剩余10%预留机动）
```

### 4.2 回调买入法

```
入场条件（全部满足）：
1. 收盘价 > MA(CLOSE, 60) 且 MA60拐头向上
2. 股价回调至 MA20 或 MA60 附近（偏差 ±2%）
3. 回调缩量：MA(VOL, 5) < MA(VOL, 20) × 0.7
4. 出现企稳K线（锤子线/看涨吞没/十字星）
5. 盈亏比 ≥ 2:1

仓位分配（金字塔加仓）：
  首批：计划仓位 50%
  确认加仓：放量反弹突破回调前高 → 加仓 30%（剩余20%预留机动）
```

### 4.3 底部右侧确认买入法

```
入场条件（至少满足5项，其中1-3为必选项）：

必选项：
1. 股价从高点跌幅 > 40%
2. 出现明确底部形态（早晨之星/底部吞没/W底突破颈线）
3. 大盘同期见底企稳（至少不再创新低）

加分项（至少满足2项）：
4. 成交量从地量后放量：MA(VOL,5) 从低于 MA(VOL,20)×0.5 回升至 MA(VOL,20)×1.0 以上
5. RSI(14) 从 < 30 回升至 > 35 且拐头向上
6. 出现基本面催化剂（政策利好/业绩超预期/行业拐点）
7. MACD 零轴下方金叉且柱状图连续放大

底部策略特殊限制：
  - 排除连续2年净利润为负的股票
  - 突破日成交量 > 20日均量 × 2
  - 单只最大仓位 = 总资金 3%（降低风险暴露）
  - 如果底部股票从买入价下跌 > 8%，无论ATR止损位如何，立即出场

仓位分配（分三批）：
  第一批：30%（右侧信号确认日）
  第二批：30%（突破颈线/前期阻力位）
  第三批：40%（回踩确认不破颈线）
```

### 4.4 止损规则（铁律，不可违反）

#### ATR 动态止损

```
ATR(14) = 14日平均真实波动幅度
买入价 = P
止损价 = P - N × ATR(14)

N 的取值（根据策略类型）：
  趋势突破：N = 1.5
  回调买入：N = 2.0
  底部右侧确认：N = 3.0

波动率自适应调整：
  当 ATR(14)/买入价 > 5%（高波动）：N 增大 0.5（给更多空间，避免频繁扫损）
  当 ATR(14)/买入价 < 1.5%（低波动）：N 减小 0.5（收紧止损，低波动不需要过宽空间）
```

#### 其他止损规则

| 规则 | 细节 |
|------|------|
| **技术止损** | 跌破关键支撑位（均线/前低/趋势线），取最严格者 |
| **时间止损** | 买入后 N 个交易日未按预期运行（见2.3策略表） |
| **逻辑止损** | 买入逻辑被证伪（业绩暴雷/政策转向），立即出场 |

**止损优先级：** 止损价 = min(ATR止损价, 技术止损价)
**底部策略额外限制：** 止损价 = min(ATR止损价, 技术止损价, 买入价 × 92%)（即8%硬上限）

### 4.5 止盈规则

#### ATR 动态跟踪止盈（替代固定百分比）

```
跟踪止盈价 = 持仓期间最高收盘价 - 回撤幅度
回撤幅度 = M × ATR(14)

M 的取值（根据策略类型）：
  趋势突破：M = 2.0
  回调买入：M = 3.0
  底部右侧确认：M = 4.0

波动率自适应调整（与止损相同）：
  ATR/P > 5%：M 增大 0.5（高波动给更多回撤空间）
  ATR/P < 1.5%：M 减小 0.5（低波动收紧回撤）

规则：只上不下
  每日更新：新跟踪止盈价 = max(昨日跟踪止盈价, 今日最高价 - M × ATR)
  触发：收盘价 < 跟踪止盈价 → 次日开盘卖出
```

#### 其他止盈方式

| 方法 | 规则 | 适用场景 |
|------|------|---------|
| **目标止盈** | 到达目标位卖出1/2，剩余启动跟踪止盈 | 波段行情 |
| **信号止盈** | 放量滞涨 / MACD死叉 / 顶部形态出现 | 所有场景 |
| **分批止盈** | 第一目标位出30%，第二目标位出30%，剩余跟踪止盈 | 中长线 |

#### 止盈后重新入场规则

> 止盈卖出后，不追高重新入场，等待新的信号确认。

| 场景 | 重新入场条件 | 仓位 |
|------|------------|------|
| **跟踪止盈卖出，股价回踩企稳** | 回踩至MA20附近 + 缩量企稳K线 + 盈亏比 ≥ 2:1 | 正常仓位 |
| **目标止盈卖出半仓，趋势延续** | 放量创新高 + MACD柱状图持续放大 | 剩余仓位继续跟踪止盈，不加回已卖部分 |
| **信号止盈卖出，股价重新走强** | 需等待完整的买入信号（不因"刚卖过"而放宽条件） | 正常仓位 |
| **止盈后股价持续上涨** | 不追，放弃本次机会，等待下一个标的 | — |

> **核心原则：止盈后的重新入场必须满足与首次买入相同的全部条件，绝不降低标准。**

### 4.6 盈亏比计算与买入决策树

**核心公式：盈亏比 = 目标收益空间 / 止损空间，要求 ≥ 2:1**

```
止损空间 = N × ATR(14)
目标收益空间 = 目标价 - 买入价

目标价确定：
  趋势突破：目标价 = 突破位 + (突破位 - 前期回调最低点)
  回调买入：目标价 = 前高位置 或 MA20 上方阻力位
  底部右侧：目标价 = 斐波那契 38.2% 或 50% 回撤位
```

**买入决策树：**

```
发现一只股票
  │
  ├─→ 触发排雷？ ── 是 → 跳过
  │               └─ 否 ↓
  ├─→ 基本面达标？ ── 否 → 加入观察池
  │                └─ 是 ↓
  ├─→ 大盘状态允许买入？ ── 否 → 等待
  │                      └─ 是 ↓
  ├─→ 出现买入信号？ ── 否 → 设提醒，等待
  │                   └─ 是 ↓
  ├─→ 盈亏比 ≥ 2:1？ ── 否 → 跳过
  │                   └─ 是 ↓
  ├─→ 计算仓位：单只 = min(总资金×2% / 止损幅度, 25%)
  │
  └─→ 执行买入 + 设止损 + 记录计划
```

### 4.7 A股特殊风险应对

| 风险 | 说明 | 应对规则 |
|------|------|---------|
| **T+1** | 当日买入不可卖出，盘中无法止损 | 盘中触发止损 → 次日开盘价卖出。买入时需预留次日波动风险 |
| **涨跌停(±10%/±20%)** | 跌停时止损单无法成交 | 跌停当天无法卖出 → 次日开盘价卖出，不计入"未守纪律"。科创板/创业板±20%需额外警惕 |
| **流动性** | 小盘股流动性差，滑点大 | 排除日成交量 < 500万股 或 日均成交额 < 5000万元的股票 |
| **ST/退市风险** | 连续亏损 → ST → 退市 | 已在排雷中覆盖：排除连续2年净利润为负的股票 |
| **涨跌停补充规则** | 持仓股涨停不止盈（让利润奔跑），跌停视原因处理 | 若因利空消息跌停 → 次日开盘卖出；若因系统性恐慌 → 按正常止损规则处理 |
| **连续跌停** | 连续 2 天以上跌停，实际亏损远超风险预算 | 连续跌停超过 2 天 → 恢复交易后立即市价卖出，不计入"执行纪律"违规 |

## 五、仓位管理与回撤恢复

### 5.1 总仓位控制（与市场状态绑定）

| 大盘状态 | 总仓位上限 | 策略选择 |
|---------|----------|---------|
| 牛市确认 | 80-100% | 趋势突破（主策略） |
| 偏多 | 50-70% | 回调买入（主策略） |
| 震荡 | ≤ 50% | 回调买入（减仓版） |
| 偏空 | 20-40% | 仅观望，不主动买入 |
| 熊市 | 0-20% | 空仓（底部右侧策略仅限模拟盘） |

**市场状态转换时的持仓处理：**

| 转换方向 | 持仓处理规则 |
|---------|------------|
| 牛市 → 偏多/震荡 | 不主动卖出，但停止加仓，跟踪止盈收紧（M 减小 0.5） |
| 偏多 → 偏空/熊市 | 总仓位超限时，卖出最弱持仓（亏损最大或技术面最差的）降至目标上限 |
| 震荡 → 熊市 | 仅保留浮盈持仓且止损在成本价以上的，其余全部卖出 |
| 熊市 → 偏多 | 观望确认（至少 5 个交易日七维判定稳定）后，小仓位试水回调买入 |
| 任何 → 牛市 | 新仓可用趋势突破策略，现有持仓可启动加仓规则 |

> **核心原则：状态降级时先减仓后观察，状态升级时先确认后行动。**

### 5.2 个股仓位

| 规则 | 标准 |
|------|------|
| 单只上限 | 总资金 25%（绝对硬性上限） |
| 单只标准 | 总资金 10-15%（常规） |
| 底部策略单只上限 | 总资金 3%（高风险策略特殊限制） |
| 行业集中度 | 同行业不超过总仓位 40% |
| 持仓数量 | 5-10 只 |

### 5.3 风险预算与仓位计算

> **核心原则：每笔交易的最大亏损不超过总资金 2%。**

```
单只仓位 = min(总资金 × 2% / 止损幅度, 总资金 × 策略仓位上限)

策略仓位上限：
  趋势突破 / 回调买入：25%
  底部右侧确认：3%

止损幅度 = N × ATR(14) / 买入价
（N 根据策略类型取 1.5 / 2.0 / 3.0，见 4.4 节）

高波动调整：ATR/P > 5% 时，N 增大 0.5
低波动调整：ATR/P < 1.5% 时，N 减小 0.5

示例 1（回调买入 — 止损幅度小，触及仓位上限）：
  总资金 100万，买入价 30元，ATR(14) = 1.2元
  回调买入策略（N=2.0），止损幅度 = 2.0×1.2/30 = 8%
  单只仓位 = min(100万×2%/8%, 100万×25%) = min(25万, 25万) = 25万

示例 2（趋势突破 — 止损幅度大，触及风险预算上限）：
  总资金 100万，买入价 50元，ATR(14) = 2.5元
  趋势突破策略（N=1.5），止损幅度 = 1.5×2.5/50 = 7.5%
  单只仓位 = min(100万×2%/7.5%, 100万×25%) = min(26.7万, 25万) = 25万

示例 3（底部策略 — 受策略上限约束）：
  总资金 100万，买入价 10元，ATR(14) = 0.4元
  底部策略（N=3.0），止损幅度 = 3.0×0.4/10 = 12%
  单只仓位 = min(100万×2%/12%, 100万×3%) = min(16.7万, 3万) = 3万
```

### 5.4 加减仓规则

| 操作 | 条件 | 动作 |
|------|------|------|
| **加仓** | 浮盈 > 1.5 × ATR(14) + 趋势延续 | 金字塔加仓（首批40%，加仓30%，再加20%） |
| **减仓** | 浮盈从高点回撤超50% 或 信号走弱 | 减至半仓 |
| **清仓** | 触发止损或逻辑破坏 | 全部卖出 |
| **补仓禁止** | 亏损时绝不在亏损方向加仓 | 止损而非补仓 |

### 5.5 回撤恢复协议

> **核心原则：回撤到一定程度，系统性降级，而非情绪化应对。**

| 回撤程度 | 应对措施 |
|---------|---------|
| **总回撤 5-10%** | 正常波动，继续执行系统规则 |
| **总回撤 10-15%** | 减仓至 50%，暂停新开仓，复盘最近5笔交易寻找共性问题 |
| **总回撤 15-20%** | 减仓至 20%，只保留最强持仓，全面复盘系统 |
| **总回撤 > 20%** | 立即空仓，回归模拟盘至少 2 周，重新验证系统有效性 |

**恢复加仓流程：**
```
空仓后恢复条件（全部满足才可恢复实盘）：
1. 模拟盘期间至少 2 笔盈利交易
2. 模拟盘总收益为正
3. 模拟盘严格执行了所有止损规则

恢复后初始仓位 ≤ 正常仓位的 50%
盈利 2 笔后逐步恢复正常仓位
```

## 六、交易纪律与执行

### 6.1 铁律（零容忍）

| 编号 | 规则 | 违反后果 |
|------|------|---------|
| 1 | **止损不可撤销** — 到了止损位必须卖 | 小亏变大亏 |
| 2 | **不追涨杀跌** — 没有信号不进场 | 高位接盘/低位割肉 |
| 3 | **不逆势满仓** — 熊市不加杠杆 | 爆仓/深度套牢 |
| 4 | **不在亏损仓位加仓** — 止损而非补仓 | 亏损放大 |
| 5 | **不过度交易** — 每月交易不超过 10 次 | 手续费吞噬利润 |
| 6 | **不碰不懂的票** — 不跟风听消息 | 盲目赌博 |
| 7 | **不临时起意** — 所有计划盘前制定 | 情绪化操作 |

### 6.2 交易前检查清单

```
□ 大盘状态是否适合交易？（对照 2.1 七维判定表 + 2.3 决策树）
□ 该股是否通过排雷清单？（对照 3.2）
□ 是否有明确买入信号？（对照 4.1/4.2/4.3）
□ 止损位是否已设定？（ATR计算 + 条件单）
□ 盈亏比是否 ≥ 2:1？
□ 仓位是否在风险预算内？（5.3 公式计算）
□ 是否已设条件单/提醒？（不依赖盘中判断）
□ 是否在冷静状态下决策？（不在开盘/收盘前10分钟冲动）
```

### 6.3 每日执行流程（精简版）

**盘前（8:30-9:25）：**
- 查看隔夜美股、A50期货、重大政策/公告
- 逐只检查持仓是否触发止损/止盈，更新跟踪止盈价
- 确认今日关注标的及触发价格，预设计条件单
- 写下今日禁做事项

**盘中（9:30-15:00）：**
- 开盘15分钟不操作，仅观察
- 条件单触发 → 按计划执行，不修改
- 无触发 → 不操作
- 尾盘14:30后检查是否需要调仓

**盘后（15:00后）：**
- 填写交易记录（见6.4模板）
- 更新持仓表（现价、盈亏、止损位、跟踪止盈价）
- 整理明日计划

### 6.5 心理管理

| 情绪 | 触发场景 | 应对方法 |
|------|---------|---------|
| **贪婪** | 连续盈利，想加仓加杠杆 | 回顾仓位上限规则，严格执行 |
| **恐惧** | 连续亏损，不敢下单 | 缩小仓位恢复信心，检查系统是否失效 |
| **后悔** | 卖出后继续涨 / 未买涨了 | 接受不确定性，关注过程而非结果 |
| **侥幸** | 亏损持仓期待回本 | 执行止损，市场不关心你的成本 |

### 6.6 系统失效判定与迭代

**系统失效信号（任一出现即暂停交易）：**

| 信号 | 应对 |
|------|------|
| 连续 5 次交易止损 | 检查选股/买入条件是否失效 |
| 单月亏损 > 8% | 检查仓位管理是否合规 |
| 最大回撤 > 15% | 触发回撤恢复协议（见5.5） |
| 月度胜率连续3月 < 30% | 系统可能不适应市场，暂停并重构 |
| 连续3月跑输沪深300 | 重新评估策略有效性 |

**迭代流程：**
```
月度复盘 → 识别亏损共性
    │
    ├── 执行问题（未守纪律）→ 强化规则，不改系统
    ├── 选股问题（胜率低但盈亏比正常）→ 调整筛选条件
    ├── 时机问题（胜率低且盈亏比低）→ 优化技术信号参数
    └── 市场问题（系统不适应）→ 暂停策略，切换市场状态应对方案
```

**回测要求：** 任何策略调整必须经过回测验证，至少3年数据、50次信号、每笔扣0.5%成本。
