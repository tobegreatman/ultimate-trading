## 二、通用排雷后缀

> **用法：** 以下条件视为"基础过滤"，建议与任意策略组合使用。
> 前三项（非ST/非停牌/商誉<30%）直接嵌入下方所有提示词。
> 后四项需在候选池出来后到 F10 手动核实。

**选股器内叠加（自动）：**

```
非ST且非停牌且商誉占净资产比例小于30%且大股东质押比例小于60%且上市天数大于250天且最近一期审计意见为标准无保留意见
```

**F10 手动复核（候选池出来后逐一检查）：**

| 检查项 | F10路径 | 排除标准 |
|--------|---------|---------|
| 关联交易 | 公司资料 → 关联交易 | 关联交易/营收 > 40% → 排除 |
| 存贷双高 | 资产负债表 | 货币资金/总资产>30% 且 有息负债/总资产>30% → 排除 |
| 董监高减持 | 股东研究 → 高管增减持 | 近一年累计减持 > 总股本5% → 排除 |
| 诉讼/处罚 | 公司资料 → 重大事项 | 近一年重大诉讼或监管处罚 → 排除 |

## 四、PC端自定义公式（综合选股）

> 在东方财富PC端「公式管理器」中新建条件选股公式，粘贴使用。

### 4.1 趋势突破

```
C>REF(HHV(H,20),1) AND V/REF(MA(V,20),1)>1.5
AND MACD.DIF>MACD.DEA AND MACD.MACD>0
AND MA(C,20)>MA(C,60) AND MA(C,60)>REF(MA(C,60),5);
```

### 4.2 回调到20日均线企稳

```
MA(C,5)>MA(C,10) AND MA(C,10)>MA(C,20) AND MA(C,20)>MA(C,60)
AND L<=MA(C,20)*1.02 AND C>=MA(C,20)*0.98
AND MA(V,5)<MA(V,20)*0.7
AND MA(C,60)>REF(MA(C,60),5);
```

### 4.3 回调到60日均线

```
C>MA(C,60) AND MA(C,60)>REF(MA(C,60),5)
AND L<=MA(C,60)*1.02 AND C>=MA(C,60)*0.98
AND MA(V,5)<MA(V,20)*0.7;
```

### 4.4 MACD底背离

```
LLV(L,30)=L AND LLV(MACD.DIF,30)>REF(LLV(MACD.DIF,30),5)
AND MACD.DIF>MACD.DEA;
```

### 4.5 底部放量长阳+RSI拐头

```
HHV(H,120)/C>1.4
AND C/REF(C,1)>1.03
AND V/REF(MA(V,20),1)>2
AND C>O AND (C-O)/O>0.03
AND RSI.RSI1>REF(RSI.RSI1,1) AND REF(RSI.RSI1,5)<30;
```

## 五、每日操作流程

```
1. 盘前判环境：牛市/震荡/熊市？（见交易系统 2.2 节）
2. 跑"宽松版"建池 → 5-20只候选
3. 按环境选策略：
   牛市 → 叠加趋势突破条件
   震荡 → 叠加回调买入条件
   熊转牛 → 叠加底部右侧确认条件
4. F10深度排雷（关联交易、存贷双高、减持、诉讼）
5. 计算盈亏比 ≥ 2:1 + 仓位 ≤ min(2%风险预算, 25%上限)
   （见交易系统 4.5/6.3 节）
6. 检查同行业持仓不超40%，同产业链不超3只
7. 设条件单，记录交易计划
```

---

> **提示：** 选股器是起点不是终点。出票后仍需 F10 排雷 + K线确认 + 盈亏比计算，方可纳入交易计划。
