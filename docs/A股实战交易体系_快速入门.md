## 一、30秒判断当前市场状态

**看两个数据即可：**

1. **指数 vs 250日均线（年线）**
   - 站上年线 + 年线向上 → 牛市或震荡偏多
   - 年线以下 + 年线向下 → 熊市
   - 均线缠绕 → 震荡市

2. **两市成交额**
   - >1万亿 → 活跃，可操作
   - 5000-8000亿 → 一般，谨慎操作
   - <5000亿 → 冷清，不宜重仓

## 二、仓位速查表

| 市场状态 | 总仓位 | 策略 |
|----------|--------|------|
| 明确牛市 | 60-80% | 核心+卫星持仓 |
| 震荡偏多 | 40-60% | ETF+少量个股 |
| 震荡市 | 20-40% | ETF定投+区间交易 |
| 震荡偏空 | 10-20% | 高股息防御 |
| 明确熊市 | 0-10% | 空仓+债券+黄金 |

**限制：单只股票≤20%、单一行业≤35%**

## 三、核心参数速查

### 止损
| 场景 | 标准 |
|------|------|
| 短线 | 5% |
| 中线 | **8%（默认）** |
| 长线 | 10% |
| 跌破关键支撑 | 无条件止损 |

### 止盈（趋势跟踪法）
| 信号 | 操作 |
|------|------|
| 从高点回撤 8-12% | 减仓50% |
| 从高点回撤 15-20% | 清仓 |
| 跌破60日均线 | 无条件清仓 |

### 单笔仓位计算
```
仓位 = 总资金 × 2% ÷ (买入价 - 止损价)
```

## 五、选股四重过滤（快速版）

```
□ 估值OK？  PE分位<50% 且 PEG<1.2
□ 基本面OK？ 营收增速>15% 且 净利润增速>20% 且 ROE>12%
□ 技术面OK？ 站上60日均线 且 60日均线向上
□ 资金面OK？ 近3日大单净流入 或 龙虎榜有机构买入
→ 四项全通过才买
```

## 六、逃顶信号速查

出现**3个以上★★★**信号，仓位降至10%以下：
- ★★★ 权重股拉升但个股不涨
- ★★★ 放量滞涨（天量但不涨）
- ★★★ 利好不涨（高开低走）
- ★★★ 北向资金大幅流出
- ★★☆ 融资余额暴增
- ★★☆ 垃圾股/ST股飞天
- ★★☆ 身边人都在讨论股票

## 八、策略切换触发条件

| 变化 | 触发条件 | 操作 |
|------|----------|------|
| 牛→熊 | 跌破250MA+放量 | 3天内仓位降至10% |
| 牛→震荡 | 3个以上逃顶信号 | 1周内降至40% |
| 震荡→牛 | 放量突破区间上沿 | 2周内建仓至60% |
| 熊→震荡 | 底部信号出现3个+ | 开始定投，≤30% |

> **8条铁律：**
> 1. 先判市场再选股
> 2. 估值是锚，趋势是帆
> 3. 计划你的交易，交易你的计划
> 4. 止损8%，趋势止盈
> 5. 每笔风险不超过2%
> 6. 牛市进攻，震荡定投，熊市防御
> 7. 持续复盘，不断进化
> 8. 保持情绪稳定，纪律是盈利的保障
